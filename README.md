# Gravity Visualizer

An interactive browser-based gravity and launch physics visualizer built with Three.js.

## Live Demo

https://hackertuhin.github.io/Gravity_Visualizer/

## Features

- Adjust force, mass, gravity, and bounce.
- Enter custom floating-point values directly.
- View speed, current height, estimated max height, acceleration, and bounce count.
- Reset all controls and simulation state to defaults.

## Run Locally

Open `index.html` in a browser, or serve the folder with a local static server.
